You fell out of the quote wire. This is a known edge in the machinery, not a secret page.

Direct quote visits become journeys by searching for a compact seed that makes the visited
quote the generated root. The search is bounded because this is a small curated archive,
not an infinite machine.

Usually it finds the path. This time it did not, so Elsewhere stops instead of pretending
the wire still holds.
