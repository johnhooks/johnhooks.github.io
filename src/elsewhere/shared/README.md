# Shared Elsewhere tools

Shared helpers for Elsewhere subprojects live here.

This is the utility belt for the cyber surfer: seeded randomness, color generation, motion helpers, shared CSS fragments, route helpers, and small primitives that help strange pages stay coherent without becoming tame.

Do not turn this into a generic app framework. Add shared code only after a pattern appears in more than one Elsewhere experience or clearly belongs to the whole layer.
