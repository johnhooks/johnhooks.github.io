# Elsewhere index

The Elsewhere index is the threshold page.

It should not feel like a normal archive. It is the hatch, the dial tone, the exposed wire, the nightclub door in the alley behind the blog. It introduces the visitor to Elsewhere and points toward active subprojects without explaining away all of the mystery.

Keep route files thin. Put the real component, styles, and local behavior for the index in this directory when implementation begins.
