# Elsewhere Quote Sources Plan

Elsewhere quotes should grow slowly from sources with clear attribution and reuse rights.

The goal is not to build a huge quote database. The goal is to curate enough good programming and web craft quotes to make the quote wire feel alive without creating licensing or attribution problems.

## First Source: Wikiquote

Wikiquote is the preferred source for the next import pass.

Why:

- stable source URLs
- usable source pages for programming topics and individual people
- Creative Commons Attribution-ShareAlike licensing
- straightforward attribution metadata
- enough material to expand without building a complex importer yet

Avoid random “best programming quotes” sites. They are often copied from each other, poorly attributed, and unclear on reuse rights.

## Import Standard

Each quote entry should preserve enough metadata to render attribution clearly:

- author
- work, when known
- source name
- source URL
- license
- retrieval date
- tags

Prefer a smaller number of good quotes over a large noisy collection.

## Later Sources

Possible later sources, with more care:

- Project Gutenberg / public domain books
- GNU or FSF essays with compatible licensing
- official documentation under permissive licenses
- personal blog posts with explicit reuse licenses

Do not import quote collections, blog posts, repositories, images, or API data until the reuse license is verified.

## Current Recommendation

Expand from Wikiquote individual author pages first. This keeps the collection consistent and gives the quote wire more texture without opening a licensing mess too early.
