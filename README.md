# johnhooks.io

[johnhooks.io](https://johnhooks.io)

Personal website built with [Astro](https://astro.build) and deployed to Cloudflare Workers.

## Developing

```bash
pnpm install
pnpm dev
```

## Building

```bash
pnpm build
pnpm preview
```

## Deployment

```bash
pnpm deploy
```

Or test locally with Wrangler after building:

```bash
pnpm build
pnpm wrangler dev --env staging
```
